Background

Singapore Health Services (Singhealth) is Singapore's largest group of healthcare institution. MyCare is one of the applications in SingHealth's Nursing Software Suite system development project. The application is installed on a managed iPad which is placed on the patient's bedside. It seeks to provide an improved in-ward patient experience for patients, as well as increasing nurses' overall productivity. Along the way, we continued to work closely with the Singhealth stakeholders during the development phase to collaboratively create a mobile application experience that truly address the end users' pain points.

Highlights

  • One of the biggest projects I worked on. This project honed my technical skills in terms of server development. The most interesting part of this project is being able to integrate our application into the vast amount of software systems used by the organization like the EMR and their sophisticated private cloud network.
  • Created a mockup EMR server to facilitate application development on staging.
  • During the pandemic, we quickly created a lite version of the application in order to minimize the interactions between the COVID-19 patients and the care providers.
  • The most challenging part was implementing iOS's SSO Extension to enable single login on the device to a suite of applications, not just MyCare. The challenge was another vendor was going to use the signin function but their application is not a native application but a Progressive Web App. Apple's SSO Extension works perfectly fine with native but not with PWA. We managed to create a workaround by discussing the whole SSO extension's process under the hood with Apple engineers.
