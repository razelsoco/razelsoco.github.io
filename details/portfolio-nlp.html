Project Information

  • Category: Machine Learning
  • Project Date: Dec 2022 - Mar 2023

Background

A text-to-SQL challenge using a large cross-domain dataset, Spider, and the pre-trained language model T5. This is one domain in Natural Language Processing where a plain text can be translated into an SQL query using a model using the T5 model.

Highlights

  • Worked as a research assistant for this project where I managed to setup an environment on a dedicated server in order to train the model for longer periods of time than Kaggle allows.

  • Our models got comparable results with the ones in the Spider dataset leaderboard.

  • Helped write draft of the article for this research project.

  • One of the major takeaways I got from this project is the importance of tracking the version numbers of the packages used. Updating packages could lead to non-working code, incompatibility issues with other package dependencies which was the main hurdle when they initially tried it the year before.

Technologies Used

Python, Pytorch Lightning, Ubuntu, T5
