Project Information

  • Category: Data Analysis
  • Project Date: March 2022

Background

The Covid-19 pandemic has greatly impacted our daily lives. Since March 2020, a lot of data has been collected in relation to the coronavirus to understand the factors that affect it's transmission, symptoms, mortality rate, etc. This is a class project which aims to give a descriptive analysis on the COVID-19 pandemic based on the collected data in U.S.

Highlights

  • This is a collaboration with 2 other classmates and my part was to analyze the impact social distancing to the number of covid cases. Our group mainly used SAS for the analysis; however, data manipulation in SAS is not so straightforward. Hence, I used Python to be able to plot the graphs I wanted, like the comparison of the trends between social distancing and the number of covid cases per month.

  • A thorough research was done about COVID-19 pandemic and applied different statistical methods like Spearman correlation, binning, encoding, and chi-square test to be able to derive an information from the data.

Technologies Used

SAS, Python, Jupyter
