Project Information

  • Category: Machine Learning | Data Analysis | Deployment
  • Project: Senior Well-Being Twitter Data
  • Project Date: April 2023

Background

This was our capstone project for our final term in the Data Analytics program. This was a continuation of the project done by the previous term students. Our work focuses on improving the topic model using the BERTopic model, setting proper evaluation criteria, and building a data pipeline to deploy the project as a data product.

Highlights

  • We employed the use of Topic Tuner to find optimal parameters to train the BERTopic model. The main challenge of the topic modelling was evaluating model as the quantitative metrics are not enough to select the best model.

  • Our team of 3 had to learn three technologies at the same time in two months that none of has used or worked with, such Django Web Framework for the dashboard, Docker for containerization, and Airflow for orchestration.

  • Using my knowledge in cloud services, I helped the team deploy the pipeline in Oracle cloud using Oracle Linux 8 instance.

Technologies Used

BERTopic for topic modelling, RoBERTa for sentiment analysis, Python, Django, Docker, Airflow, PostgreSQL, GitLab, HTML, CSS, Javascript, REST APIs, Windows, Linux, Oracle cloud,
